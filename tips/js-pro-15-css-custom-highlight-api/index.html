CSS Custom Highlight API
JavaScript is a versatile programming language that powers the modern web. With features like async/await, destructuring, and optional chaining, JavaScript makes complex tasks simple. The CSS Custom Highlight API lets you highlight ranges of text in the DOM without modifying its structure — combining the power of JavaScript and CSS for elegant, performant highlights.
// No DOM wrappers needed: const range = new Range() range.setStart(textNode, start) range.setEnd(textNode, end) const hl = new Highlight(range) CSS.highlights.set('search-result', hl)
